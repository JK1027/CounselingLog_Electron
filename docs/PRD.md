# 📋 제품 요구사항 정의서 (PRD.md)

> 대상 프로젝트: Electron + React + Python FastAPI 기반 상담일지 현대화 앱
> (기존 Tkinter 앱의 UX/UI 완전 재설계 + 엑셀 호환 유지)

---

## 1. 프로젝트 개요 및 배경

기존 Tkinter 기반의 상담일지 작성 프로그램(`Counseling_Log`)의 노후화된 UX/UI를 개편하여, 학교 상담 교사의 업무 생산성을 극대화하기 위해 개발된 현대적인 데스크톱 애플리케이션입니다. 
* **핵심 컨셉**: "엑셀은 저장 포맷일 뿐이며, 앱은 실제 작업 공간이다."
* **사용자 타겟**: 초·중·고등학교 상담 교사 (엑셀 파일 직접 편집의 번거로움과 데이터 유실 리스크 최소화)

---

## 2. 엑셀 호환성 제약 (절대 원칙)

본 애플리케이션은 기존 엑셀 데이터 파일과의 100% 호환성을 보장해야 합니다. 다음 항목은 어떤 경우에도 임의로 변경할 수 없습니다.

### 2-1. 기존 시트명 및 스키마
* **개인상담**: 일자, 학년, 학번, 성별, 이름, 회기, 상담구분, 상담방법, 제목, 상담내용, _record_id
* **집단상담(또래상담, 학급별 집단)**: 일자, 학년, 반, 구분, 회기, 주제, 활동내용, 참가학생(이름), 참가학생(학번), _record_id
* **보호자상담**: 일자, 학년, 학번, 성별, 이름, 대상, 상담구분, 상담방법, 제목, 상담내용, _record_id
* **교원자문**: 일자, 학년, 학번, 성별, 이름, 대상, 제목, 상담내용, _record_id
* **의뢰(정서행동의뢰, 자문의 의뢰 등)**: 일자, 학년, 학번, 성별, 이름, 의뢰사유, 조치결과, 비고, _record_id

### 2-2. 엑셀 구조 제약 사항
* **고유 식별자 (`_record_id`)**: 각 시트의 상담 데이터 행을 정확히 식별하여 수정 및 삭제하기 위해 UUID 형식의 고유 ID 컬럼을 맨 끝에 추가하되, MS Excel 등으로 열었을 때 노출되지 않도록 **열 숨김(Hidden) 처리**를 수행합니다.
* **서식 보존**: 프로그램 구동 시 엑셀의 원본 스타일(셀 크기, 병합, 테두리, 글꼴, 인쇄 영역 설정 등)을 훼손하지 않아야 하며, openpyxl을 통해 신규 행을 추가할 때도 기본 정렬 및 테두리 서식을 유지합니다.
* **나이스(NEIS) 업로드 구조**: 교육행정정보시스템(나이스) 업로드 시 데이터 순서 및 컬럼 구조가 어긋나지 않도록 엑셀 구조 변경을 철저히 금지합니다.

## 3. 핵심 기능 명세 (Feature Spec)

### 3-1. 학생 및 상담 관리 (CRUD)
* **조회 및 통계**: 오늘 상담 건수, 주의가 필요한 학생 목록, 최근 상담 기록을 한눈에 볼 수 있는 메인 대시보드 제공.
* **상담 등록 및 수정**: 상담 일자, 유형(시트), 구분, 제목, 내용 등을 입력하는 폼 에디터(`QuickEditor`) 제공. 중복 저장 방지를 위한 비활성화 가드 및 로딩 상태 제공.
* **상담 삭제**: `_record_id` 기반 데이터 삭제 및 삭제 후 엑셀 내 순번 재정렬 기능 지원. 롤백 대비 Safe Save 매커니즘 필수 적용.
* **학생 정보 수정**: Timeline 헤더에서 학생의 이름, 학번(4자리), 학년, 성별을 일괄 수정하고 관련된 모든 시트의 기록을 원자적으로 업데이트.

### 3-2. UX 및 입력 생산성
* **초성 및 학번 검색**: 자음 초성 검색(예: `ㄱㅁㅅ` ➔ `김민수`) 및 학번 검색을 지원하는 초고속 Command Palette(단축키 `Ctrl+K`).
* **연속 입력 모드**: 상담을 완료한 직후 다음 학생으로 자동 포커스되어 흐름이 끊기지 않고 연속 작성을 이어나갈 수 있는 기능.
* **단축키 시스템**: `Ctrl+N`(새 상담), `Ctrl+S`(상담 저장), `Esc`(에디터 닫기), `Alt+ArrowUp/Down`(학생 빠른 이동) 등 키보드 중심의 동선 제공.
* **컴팩트 모드 (Compact View)**: 화면 노출 밀도를 극대화하여 25% 축소된 레이아웃 및 1줄 접기 본문 제공.
* **Resizable Split Pane**: 마우스 드래그를 통해 사이드바 및 에디터의 좌우 너비를 실시간으로 개인화하고, 세션 간 조절 수치를 복원(안전 범위: 사이드바 200~400px, 에디터 280~500px).

### 3-3. 파일 관리 및 드래그 앤 드롭
* **다이얼로그 파일 탐색**: 상단 메뉴 `파일 > 엑셀 파일 열기(Ctrl+O)`를 통해 로컬 경로에 있는 다양한 상담일지 엑셀 파일 로딩.
* **드래그 앤 드롭 파일 로드**: 외부 엑셀 파일을 앱 어느 영역에든 끌어다 놓으면(Drag & Drop) 흐림 오버레이 안내 후 즉시 해당 엑셀의 데이터를 읽어와 화면 갱신.

### 3-4. 다각도 인쇄 및 PDF 저장
* **인쇄 설정**: 선택된 학생의 전체/회기별 출력, 시트 유형별 전체 출력 등 유연한 범위 필터링 제공.
* **인쇄 템플릿**: A4 규격을 충족하는 격식 있는 한국 교육기관 표준 상담일지 양식(상세 보고서) 및 엑셀 리스트 요약 양식(대장 양식) 지원.
* **미디어 쿼리**: `@media print` 스타일을 통해 화면 UI 요소를 숨기고 깨끗한 하얀색 A4 양식만 출력하거나 PDF로 저장 가능하도록 제어.
